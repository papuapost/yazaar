Planet Yazaar creates and maintains community-supported additions to the Yahoo User Interface (YUI) Library (Yahoo + Bazaar = Yazaar).

Compared with 'full throttle' open source projects, YUI lacks several key elements:

  • A vehicle for accepting contributor extensions, either in the form of code or documentation
  • Internal development documentation, such as coding style guides and build systems
  • A public repository with version to version change logs

The purpose of Planet Yazaar is fill in the gaps and act as a community-based adapter for the YUI Library. Our wish is for the Yazaar community to enjoy the best of both worlds: the cathedral and the bazaar!
